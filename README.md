# Deepfake Face Detection

This project detects deepfake videos using deep learning techniques.  
The system processes video frames and classifies them as **Real** or **Fake** using a PyTorch-based model.

## Project Overview
Deepfake technology can manipulate videos by replacing faces with synthetic ones.  
This project aims to detect such manipulations using frame-level analysis.

## Features
- Extract frames from input video
- Preprocess frames using image transformations
- Detect deepfake frames using a neural network
- Classify frames as **REAL** or **FAKE**
- Generate annotated output video

## Technologies Used
- Python
- PyTorch
- OpenCV
- NumPy
- ImageIO
- Google Colab

## Workflow
1. Load video frames
2. Preprocess frames
3. Run deepfake detection model
4. Classify frames
5. Annotate output video

## Output
- Total frames processed
- Number of fake frames detected
- Number of real frames detected
- Annotated output video
